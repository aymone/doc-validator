## Getting Started

First run (Install)
```bash
$ make install
```

Up containers and run project
```bash
$ make up
```

Stop containers
```bash
$ make stop
```

Read logs from containers
```bash
$ make logs
```

Update dependencies on containers
```bash
$ make deps
```

Running unit tests
```bash
$ make test
```

## Requirements

[Docker](https://www.docker.com/)
[Docker compose](https://docs.docker.com/compose/install/)
